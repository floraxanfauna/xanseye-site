# XansEye Brand Style Rules

These rules define the visual system for xanseye.com.

## Core Direction

- Minimal, white-first layout with generous negative space
- Centered hero composition and restrained page structure
- Uppercase navigation with wide letter spacing
- Soft grayscale palette with very light borders
- Editorial photography takes priority over ornament
- Simple typography hierarchy with strong readability

## Color Rules

- Background: pure white or near-white only
- Text: dark charcoal gray, not pure black
- Secondary text: medium gray
- Borders and dividers: very light gray
- Active navigation: pale gray
- Buttons: if used, keep them understated and text-led

## Spacing Rules

- Use wide outer margins and centered content columns
- Prefer large vertical spacing between major sections
- Use thin 1px divider lines between major blocks
- Avoid crowded layouts and stacked UI chrome
- Keep image and text areas visually separated

## Typography Rules

- Use a clean sans-serif for navigation and body text
- Use slightly larger, heavier text for key headlines
- Use centered intro text where the page starts with a statement
- Keep capitalization consistent and simple

## Layout Rules

- Center the logo at the top of the page
- Place a thin divider line below the logo area
- Navigation runs horizontally below the divider
- Main content should feel spacious and editorial
- Use image-led compositions with text kept secondary

## Photography Rules

- Use large, full-width or near-full-width imagery
- Prefer image strips, grids, and single strong featured shots
- Keep thumbnails compact and evenly spaced
- Avoid decorative overlays unless they are subtle and necessary

## Interaction Rules

- Hover states should be minimal
- Active page state should be light gray, not bold or colorful
- Avoid heavy shadows, gradients, and large rounded corners
- Avoid loud buttons unless a page truly needs a call to action

## Implementation Rule

- Every page update should preserve this same calm, editorial, white-space-heavy visual language.
